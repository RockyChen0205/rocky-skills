# Output Defaults

## Always-on files

### `plan.md`

最少包含：
- Goal
- Scope
- Phases
- Agent assignments
- Current status
- Decisions made

### `notes.md`

最少包含：
- Index discovery findings
- Candidate clusters
- Analyst summaries
- Final synthesis fragments

## Default final deliverable

当用户没有明确指定最终文件时：
- 默认写 `survey.md`

`survey.md` 推荐结构：
- 研究问题
- 范围说明
- 分类法
- 代表工作与对比
- 研究空白
- 可直接用于下一步的结论

## User-specified deliverables

如果用户指定了最终用途或文件名，优先服从用户，不再强制 `survey.md`。

常见例子：
- `solution-landscape.md`
- `research-gap.md`
- `proposal-input.md`
- `experiment-design-input.md`
- `reading-shortlist.md`

## Optional outputs

除非用户明确要求，否则不要默认创建：
- 大表格文件
- 逐篇卡片文件
- 单独的 shortlist 文件
- 额外的附录文件
